# notes-app

A note taking application that features,
- Creating
- Reading
- Updating 
- Deleting 
- SortBy Date created, Last edited, Alphabetical

## Motivation 

This project was created to show a CRUD application using basic javascript. 

## Built with

Javascript & express

*Installation*
 - NPM install
 - NPM run start - Node
 - NPM run dev - Nodemon
